\# Sistema de Pedido de Comida



Aplicação console que simula um sistema de pedidos de comida desenvolvida em C#.



\## Tecnologias



\- C# / .NET 8

\- Console Application



\## Funcionalidades



\- Cadastro de produtos com controle de estoque

\- Cadastro de clientes

\- Criação de pedidos por cliente

\- Adição de itens ao pedido com controle de estoque

\- Listagem de pedidos com itens e total

\- Cancelamento de pedido com devolução automática de estoque



\## Como rodar



1\. Clone o repositório

2\. Execute o projeto: `dotnet run`
